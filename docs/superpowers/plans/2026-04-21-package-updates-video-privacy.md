# Package Updates + Video Privacy Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Update all packages to latest Expo-compatible versions and ensure no video URI is ever persisted to AsyncStorage history.

**Architecture:** Two independent changes — (1) run Expo's canonical package-check tooling and apply upgrades, (2) remove the `videoUri` field from `AnalysisResult` before it is saved so no file path leaks into stored history.

**Tech Stack:** Expo SDK ~54, React Native 0.81.x, TypeScript, AsyncStorage, Gemini inline-base64 API
